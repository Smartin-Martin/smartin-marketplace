# Smartin Compass — free Claude Code & Claude Cowork plugin

Smartin Compass is a free, local-first plugin for Claude Code and Claude Cowork. It catches raw ideas as they come, sorts them under the hood, shows which ones are worth checking, and turns the week into one clean Finding worth forwarding. It is built on the Smartin Compass six-tile method for checking whether an idea has real foundations.

## Install

```
/plugin marketplace add Smartin-Martin/smartin-marketplace
/plugin install smartin-compass
```

Reload plugins if prompted with `/reload-plugins`.

## Commands

The plugin adds four commands. `/compass` shows the state of your collected ideas and what is worth checking next. `/finding` produces this week's Finding, one clean page worth forwarding. `/kompas` generates a full Smartin Compass workshop set for one mature idea. `/kompas-review` reviews an existing workshop set and flags issues.

## Skills

Two skills load automatically. The napadovnik skill catches half-thoughts, links, and customer quotes from conversation and quietly matures them. The kompas-methodology skill applies the six-tile method for checking whether an idea, segment, pricing model, or new direction has real foundations.

## Why Smartin Compass

Free and local-first, so your ideas stay in your own files. Zero-friction capture: drop a thought and it gets filed and matured under the hood. One weekly Finding instead of a pile of scattered notes.
